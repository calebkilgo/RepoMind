About RepoMind

RepoMind turns any GitHub repository into an interactive knowledge graph. Paste a URL, and it fetches the source, parses it with tree-sitter WASM, builds a call graph, and renders it with Sigma.js — all in your browser.

No server, no API key required, no code leaves your machine. Works with JavaScript, TypeScript, Python, C, and C++.
